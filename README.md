32-bit Operating System in assembly.

## Building

```
$ sudo apt install build-essential llvm lld nasm mtools
$ git clone git@github.com:d0p1s4m4/StupidOS.git
$ cd StupidOS
$ make
```

Pretty easy, isn't it?
